## github actions

you can deploy CI by github actions.

you should only push this to your repositories.

### envirnment vars

in `.github/workflows/main.yml`:

- GITHUB_BRANCH
- GITHUB_USERNAME
- GITHUB_USEREMAIL
- MD_NAME
- PDF_NAME
- TEMPLATE_NAME

edit above environment vars.
